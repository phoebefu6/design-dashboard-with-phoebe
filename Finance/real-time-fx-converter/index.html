How it behaves

Type values like 1 SGD or 100 SGD, change source/target currencies, and results recalculate instantly.

Common examples are preloaded: SGD → CNY, SGD → MYR, and USD → SGD. “MGR” is treated as “MYR” because MYR is the ISO currency code for Malaysian Ringgit.

For production, use a licensed market-data provider or backend proxy. Do not scrape Google directly from the client.
Rates are indicative and for design demonstration only.
