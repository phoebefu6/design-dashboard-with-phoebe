# FX Motion Real-time Converter

A one-page motion-led exchange-rate calculator inspired by Dobre Agency’s dark editorial motion style.

## Live dashboard

[View FX Motion Real-time Converter](https://phoebefu6.github.io/design-dashboard-with-phoebe/Finance/real-time-fx-converter/)

## Features

- Three currency conversion routes on one screen
- Editable amount per route
- Supports SGD, CNY, MYR, USD, EUR, JPY, HKD, AUD, GBP, THB, IDR, PHP, KRW, and VND
- Latest daily FX rates via a public exchange-rate endpoint
- Rate date shown in each conversion row
- Google lookup link per route for quick audit
- Motion-led one-page homepage layout

Note: Google exchange-rate results cannot be reliably crawled directly from a static browser page because of cross-origin and anti-scraping restrictions. For production, use a licensed FX market-data provider or backend proxy.
