# Design Dashboard with Phoebe

A growing collection of industry-focused dashboard concepts designed and built by Phoebe.

Each project explores how information hierarchy, interaction design, responsive behavior, and visual systems can turn complex operational data into a clear decision-making workspace.

## Projects

### Ecommerce

- [Mega Campaign Live Dashboard](./Ecommerce/mega-campaign-live-dashboard/)  
  A responsive command center for monitoring a Double Day ecommerce campaign across traffic, conversion, GMV, orders, category performance, promotions, pricing, customer-funnel drop-offs, and platform incidents.

## Design principles

- Evidence and operational context remain visible without hover.
- Desktop and mobile are treated as sibling experiences.
- Day and night themes preserve the same information hierarchy.
- Alerts use dedicated semantic colors rather than decorative emphasis.
- Filters update the full analytical state, not just isolated widgets.

## Running a project

Each dashboard is a self-contained HTML, CSS, and JavaScript project. Open its `index.html` file directly or serve the repository locally:

```bash
python3 -m http.server 8080
```

Then open the relevant project path in your browser.

All data shown in these projects is illustrative.
