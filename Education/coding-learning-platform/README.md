# CodeNest Python Learning Studio

A client-viewable concept for an interactive coding learning platform. The prototype guides beginners through a Python learning path and gives them two practice environments:

- a PyCharm-style IDE workspace for script-based practice;
- a Jupyter-style notebook workspace for notes, cells, experiments, and reflection.

## Live view

[Open the live platform](https://phoebefu6.github.io/design-dashboard-with-phoebe/Education/coding-learning-platform/)

## What is included

- Guided Python curriculum modules from setup to mini-capstone.
- Interactive module status filter.
- Safe browser-based code-run simulation for beginner Python exercises.
- Auto-check panel for variables, functions, return statements, output, and readiness score.
- Notebook-style runnable learning cells.
- AI tutor guidance and progress analytics.
- Responsive layout for desktop and mobile review.

## Notes

This version is a static GitHub Pages prototype. It does not execute arbitrary Python on the client or server. A production version can integrate Pyodide for real in-browser Python execution, or connect to a sandboxed backend runner for advanced exercises.
