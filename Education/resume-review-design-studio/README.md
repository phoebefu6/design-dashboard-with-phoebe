# Resume Atelier AI Review Studio

A client-viewable prototype for reviewing, rewriting, designing, and exporting resumes.

## Live view

[Open the live website](https://phoebefu6.github.io/design-dashboard-with-phoebe/Education/resume-review-design-studio/)

## What is included

- Resume intake through upload metadata or typed/pasted experience.
- AI-style review engine that highlights weak resume content.
- Explanation for each suggested revision.
- Accept / reject controls for every suggestion.
- Generated resume content based on approved suggestions.
- Five ATS-aware resume design formats:
  - Minimal Editorial
  - Executive Serif
  - Modern Product
  - Creative Accent
  - Compact ATS
- Browser-based PDF export using a resume-only print layout.

## Notes

This is a static GitHub Pages prototype. TXT/MD uploads can be read directly in the browser. DOC, DOCX, and PDF parsing would require a production parser service. The current PDF export opens the browser print dialog; users can choose **Save as PDF**.
